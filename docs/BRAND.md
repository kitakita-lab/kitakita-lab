# KitaKita Lab ブランドガイド

> このドキュメントは KitaKita Lab のすべての言葉・企画・デザインの拠りどころです。
> サイトのコピー、SNS、プレスリリース、企画書——迷ったら、ここに帰ってきてください。
> 5年後、10年後もぶれない「軸」として育てていきます（変更はPRで議論のうえ行う）。

---

## ブランドフィロソフィー（土台となる思想）

**挑戦は、待つものではなく、育てるもの。**

- これはキャッチコピーではなく、KitaKita Lab の**行動原理**。
- 「いつかチャンスが来たら」ではなく、目の前にやってきた小さな種を、水をやり、試し、かたちにする。
- サイトやコピーでこの一文を乱用しない。**言葉として掲げるのは要所だけ**。それ以外の場所では、この考え方が「にじみ出る」表現を選ぶ。

## 存在意義（Purpose）

**北海道から、新しい挑戦を育てる。**

KitaKita Lab は「何をしている会社か」で語られるブランドではない。
**どんな価値観で挑戦を育てるか**で語られるブランドである。

ここでいう「挑戦」は、人・地域・企業・作家・アイデア・商品・サービス・働き方——すべてを含む。
ハンドメイドから始まったが、ハンドメイドに閉じない。

## Mission（果たす役割）

**やってきた「きた」を見過ごさず、試し、育て、かたちにして社会へ届ける。**

挑戦は、ある日突然やってくる。
出店の話がきた。新しい仕事がきた。仲間がきた。アイデアがきた。
その瞬間に「育てられる場所」があるかどうかで、未来は変わる。
KitaKita Lab はその場所になる。

## Vision（目指す景色）

1. **挑戦が、日常になる** — 「やってみたい」が特別なことではなく、あたりまえに芽を出す毎日へ。
2. **北海道が、挑戦の産地になる** — 食や自然だけじゃない。「挑戦」も北海道の名産にする。
3. **「きた！」が、循環する** — 誰かの挑戦が実り、それが次の誰かの「きた」になる。喜びがめぐる生態系へ。

## Value（行動指針）

| Value | 意味 |
| --- | --- |
| **小さな「きた」を拾う** | 大きな実績より、目の前の小さな兆しを大切にする。 |
| **まず、試す** | 完璧な計画より、小さな実験。Labらしく、やってみてから考える。 |
| **挑戦を、ひとりにしない** | 一歩を踏み出す怖さを知っているから、隣を歩く。 |
| **かたちにして、届ける** | 育てた挑戦は、社会に届いてはじめて実る。最後まで伴走する。 |
| **喜びを、次の種にする** | 「きた！」の喜びを循環させる。成功は独り占めしない。 |

## 「きたきた」に込めた意味

ブランド名「KitaKita」には、4つの「きた」が重なっている。

1. **北**海道の「きた」 — 私たちの出発点であり、帰る場所。
2. **やってきた**の「きた」 — ここまで歩いてきた道のり。積み重ねてきた時間。
3. **「きた！」** — 作品が届いた日の、あの弾む気持ち。
4. **チャンスがきた**の「きた」 — 出店の話がきた。仲間がきた。アイデアがきた。未来がきた。

コピーを書くとき、この4つのどれかが自然に響く言葉を選ぶ。
4つすべてを毎回説明しない。**受け手が自分の「きた」を思い出せる余白**を残す。

## 「Lab」に込めた意味

Lab は研究所ではない。**挑戦を試し、育て、かたちにする実験室**。

- 完成したものを展示する場所ではなく、未完成を歓迎する場所。
- 「失敗」は実験の結果のひとつであって、終わりではない。
- だから活動は常に「育成中」。増えること・変わることが前提。

## 文章トーン

**読み終わったあと、少し前向きになれること。** それがすべての文章のゴール。

### トーンの要素
- **温かさ** — 手ざわりのある言葉。人の顔が浮かぶ具体。
- **希望と未来** — 課題を語るときも、必ず「ここからどうなるか」で終える。
- **北海道らしさ** — 雪解け、芽吹き、季節、産地。土地の言葉を借りる（多用はしない）。
- **余白** — 説明しすぎない。短い文。読点のリズム。受け手が考える隙間を残す。
- **誠実さ** — 大きく見せない。できないことは書かない。ダミーはダミーと明記する。

### 文体ルール
- 一文は短く。体言止めと「——」を効果的に。
- 漢字をひらきすぎない・閉じすぎない（形→かたち、育てる→そのまま）。
- 「弊社」「ソリューション」「シナジー」等の企業語は使わない。
- 感嘆符は「きた！」以外では原則使わない。
- 主語は「私たち」。読み手への呼びかけは「あなた」。

## 今後コピーを書くときの判断基準（チェックリスト）

新しいコピー・文章は、公開前にこの5問に通す。

1. **「何をやるか」ではなく「どう育てるか」が伝わるか？**
2. **4つの「きた」のどれかに、自然につながっているか？**（こじつけなら外す）
3. **読み終わって、少し前向きになれるか？**
4. **説明しすぎていないか？ 余白はあるか？**
5. **5年後に読んでも恥ずかしくないか？**（流行語・誇張・数字の盛りに頼っていないか）

3つ以上 No なら書き直す。迷ったらフィロソフィーに帰る——
**挑戦は、待つものではなく、育てるもの。**
